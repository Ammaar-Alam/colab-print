# Screenshots

Use this folder for release, social, and Chrome Web Store screenshots.

Recommended assets:

- `popup.png` — the extension popup on a supported Colab tab
- `export.png` — the export page after a successful build
- `demo.gif` — a short capture of the full flow from popup to download

Guidelines:

- Capture at 2x / retina for crispness; target widths around 720–1200 px for the PNGs.
- Keep anything identifiable (student name, drive folder, API keys in a cell, etc.) out of frame.
- Use light theme for the primary screenshot, dark theme for the secondary — it shows off theme-aware backgrounds.
